The Core Innovation

The Pharos Intelligence Layer

A backend service that monitors every client's data, detects trends, compares against industry benchmarks, and generates proactive alerts — automatically, 24/7.

1
Data Collection
Scheduled service pulls key metrics from each client's GHL sub-account every 6 hours. Contacts, pipeline, appointments, campaigns, reviews, calls — stored as historical time-series data.
GHL API v2 → Node.js service → PostgreSQL
2
Analysis Engine
Rules engine compares current vs. previous periods, flags trends (good and bad), checks against industry benchmarks, and identifies actionable opportunities. Industry templates define which metrics matter per vertical.
Trend detection + Industry-specific benchmark configs + Threshold rules
3
AI-Powered Insight Generation
Raw analysis is fed to Claude API, which generates plain-language alerts in the client's industry context. "Your appointment no-show rate ticked up 12% — we've prepared a fix" not "metric_noshowrate: +0.12 delta."
Claude API (Haiku) → Natural language alerts → ~$3/mo for 50 clients
4
Alert Delivery
Insights delivered via custom GHL dashboard widgets, email, or SMS depending on severity and tier. Positive trends update the dashboard; critical drops trigger immediate notifications.
GHL dashboard widgets + GHL messaging (email/SMS) + Custom notification logic
